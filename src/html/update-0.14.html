Welcome to QNote Thunderbird extension release 0.14.0!

Extension to attach notes to emails

Changelog:

To gain control over the code, this is a complete rewrite in TypeScript, fixing most of the issues with TB128.

  • Feature: take note screenshots to clipboard
  • Fix: note window not scrolling with large text (#48)
  • Fix: MacOS blank window (#58)
  • Fix: brought back missing text column (#57)
  • Deprecated: storage option inside the extension. This feature will be removed at some point. Please export notes into a folder.

Support the development
