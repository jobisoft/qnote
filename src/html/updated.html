Support the development

It was fun experimenting with Thunderbird (TB) internals when I first decided to rewrite XNote++ to support the latest TB version about four years ago. This was mainly because I was an XNote++ user myself and was disappointed when it stopped working due to changes in TB internals.

And now, again, some breaking changes have been made to TB internals, so much so that another rewrite was required to support the latest TB versions.
