Welcome to QNote Thunderbird extension!

Extension to attach notes to emails

Quick usage:

  • Set up storage folder and other preferences
  • Alt+Q to toggle note
  • ESC to close note without saving

Complete list of features and more information available on GitHub

Found a bug or missing a feature?

  • Check for open or closed issues, or open a new one!

It was fun experimenting with Thunderbird (TB) internals when I first decided to rewrite XNote++ to support the latest TB version about four years ago. This was mainly because I was an XNote++ user myself and was disappointed when it stopped working due to changes in TB internals.
